Since finding the site [terminaltrove](https://terminaltrove.com/) I have realised that I have accumulated a lot of CLI tools. Tools that help me in my workflow day to day or tools that are just awesome to use.

Below are a list of ones that I find _useful_

<!--truncate-->

[Note]
Any thing missing, let me know. I am always on the look for new CLI / TUI tools.

## The Must Haves

### Utilities

- [fzf](https://github.com/junegunn/fzf) - Command-line fuzzy finding written in Go
- [bat](https://github.com/sharkdp/bat) - Clone of cat(1) with syntax highlighting and Git integration
- [atuin](https://github.com/atuinsh/atuin) - Improved shell history for zsh, bash, fish and nushell
- [zoxide](https://github.com/ajeetdsouza/zoxide) - shell extension to navigate your file-system faster
- [yazi](https://github.com/ajeetdsouza/yazi) - Blazing fast terminal file manager written in Rust, based on async I/O
- [eza](https://github.com/eza-community/eza) - Modern, maintained replacement for ls
- [ripgrep](https://github.com/BurntSushi/ripgrep) - Fast search tool

### Git and GitHub

- [lazygit](https://github.com/jesseduffield/lazygit) - A simple terminal UI for git commands, written in Go
- [gh](https://github.com/cli/cli) - GitHub command-line tool
- [gh-dash](https://github.com/dlvhdr/gh-dash) - A rich terminal UI for GitHub PRs that doesn't break your flow.
- [worktrunk](https://github.com/max-sixty/worktrunk) - Git worktree management, designed for parallel AI agent workflows

### Process Management

- [zellij](https://github.com/zellij-org/zellij) - terminal multiplexer
- [lazydocker](https://github.com/jesseduffield/lazydocker) - Lazier way to manage everything docker
- [k9s](https://github.com/derailed/k9s) - kubernetes CLI to manage your clusters in style!
- [bpytop](https://github.com/aristocratos/bpytop) - Linux/OSX/FreeBSD resource monitor

## The Rest

- [bucky](https://github.com/mrpbennett/bucky) - My **very own** S3/SFTP/GCP TUI
- [mole](https://github.com/tw93/Mole) - CLI for Deep cleaning and optimizing your Mac.
- [csvlens](https://github.com/YS-L/csvlens) - Command line csv viewer
- [carapace](https://github.com/carapace-sh/carapace-bin) - CLI tool for multi-shell completion.
- [posting](https://github.com/darrenburns/posting) - A modern API client that lives in your terminal.
- [jnv](https://github.com/ynqa/jnv) - Interactive JSON filter using jq

Please do let me know if I have missed anything, you can find me on X at [@mrpbennett](https://x.com/mrpbennett)
